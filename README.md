## 📌 О проекте
Данный проект посвящен исследовательскому анализу данных (EDA) студентов онлайн-платформы. Цель работы — выявить ключевые факторы, влияющие на риск отчисления, и составить поведенческие профили учащихся для предотвращения оттока.

## 🗂️ Источник данных
Для данного проекта используется **датасет:** [Online Learning Engagement and Dropout Risk](https://www.kaggle.com/datasets/jayjoshi37/online-learning-engagement-and-dropout-risk/data)  с Kaggle о вовлеченности в онлайн-обучение и риске отсева.


## 🎯 Цели исследования
1. Общий портрет аудитории: Проанализировать распределение студентов по различным категориям.

2. Поведенческий анализ: Оценить, как ключевые метрики связаны с успеваемостью и риском отчисления.

3. Анализ по категориям курсов: Сравнить популярность, успеваемость и уровень прогресса между разными направлениями.

4. Корреляционный анализ: Определить, какие факторы наиболее сильно коррелируют с риском отсева.
   

## 📊 Этапы анализа
1. Загрузка и первичный осмотр данных: Импорт библиотек, чтение CSV-файла, вывод первых строк для ознакомления со структурой данных.

2. Предобработка данных: Преобразование текстового признака dropout_risk (Low, Medium, High) в числовой формат для удобства корреляционного анализа. Проверка на наличие пропусков (в датасете пропусков не обнаружено😇).

3. Анализ общего портрета аудитории: Исследование распределения по риску отсева и типам устройств с помощью группировок и круговых диаграмм.

4. Анализ по категориям курсов: Сравнение направлений по количеству студентов, общему времени, среднему баллу и прогрессу. Визуализация с помощью столбчатых диаграмм.

5. Анализ по группам риска: Вычисление средних значений ключевых метрик для каждой группы (High, Medium, Low).

6. Корреляционный анализ: Расчет корреляции числовых признаков с целевой переменной dropout_risk_numeric для выявления наиболее влиятельных факторов. Визуализация результатов на столбчатой диаграмме.

## Итоговое резюме
Студент с низким риском отсева — это человек, который активно и регулярно взаимодействует с платформой (часто заходит, смотрит видео), но при этом может показывать средние результаты в тестах и не спешить с прохождением материала.

Студент с высоким риском отсева, напротив, может быть "целевиком", который приходит за конкретными знаниями, успешно сдает тесты, быстро продвигается по курсу, но мало взаимодействует с остальным контентом (форумы, видео).

## ❗️ Основная рекомендация

Необходимо внедрить систему уведомлений и поддержки для "тихих", но успевающих студентов, чтобы выяснить их истинные цели и мотивировать на завершение курса. Также стоит пересмотреть сложность и сбалансированность тестов, чтобы убедиться, что высокие баллы являются следствием глубокого изучения материала, а не уже имеющихся знаний.
